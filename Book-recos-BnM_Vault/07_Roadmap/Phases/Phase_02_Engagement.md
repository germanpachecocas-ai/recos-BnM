## Objetivo de la fase

Mejorar la retención con señal de uso real. Meta: D7 ≥ 35%.

---

## Features planificadas

| Feature | Descripción | Prioridad estimada |
|---|---|---|
| Señal de afinidad histórica | Ajustar score del feed con historial de likes del usuario (`+peso a géneros con mayor ratio de like`) | P1 |
| Listas compartibles | Compartir listas de la biblioteca con otros usuarios por URL | P2 |
| Notificaciones push | Push de nuevos contenidos en géneros favoritos del usuario | P2 |
| Analytics Summary Agent | Agente que genera un reporte semanal de métricas (lunes) | P3 |
| Buscador de contenido | Búsqueda directa por título, autor o director | P1 |
| Pantalla "About" con TMDB logo | Sección de créditos con logo oficial de TMDB + atribución completa | P3 |
